# Classifying Malignant from Benign Tumors in Breast Cancer Patients
This project developed and evaluated several deep learning models to classify breast cancer histopathology images as benign or malignant using the Breast Cancer Histopathological Database. The dataset consisted of approximately 7,909 microscopic tissue images collected from 82 patients at different magnification levels. An initial convolutional neural network (CNN) model was trained using stratified sampling to address class imbalance. The model was subsequently enhanced by incorporating batch normalization, the Adam optimizer, and an exponential learning-rate scheduler to improve training stability and predictive performance. Additional experiments were conducted using data augmentation techniques such as image flipping, cropping, rotation, and color transformations. Comparative analysis showed that the batch-normalized CNN combined with the Adam optimizer and learning-rate scheduler achieved the best performance, reaching an accuracy of 86.22%. The results also indicated that some augmentation techniques may negatively affect classification accuracy by altering important diagnostic features in medical images. Overall, the study demonstrates the potential of deep learning for automated breast cancer diagnosis and suggests that future improvements could be achieved through transfer learning and more advanced neural network architectures.
